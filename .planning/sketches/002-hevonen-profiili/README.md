# Sketch 002: Hevosen profiilisivu

## Designkysymys
Mikä rakenne tuntuu parhaalta hevosen profiilisivulle, jossa on perustiedot, kilpailutulokset, sukutaulu ja kuvagalleria?

## Miten avaat
`.planning/sketches/002-hevonen-profiili/index.html` — avaa selaimessa

## Variantit
- **A: Lineaarinen scroll** — hero-palkki ylhäällä, kaikki sisältö allekkain sektioina
- **B: Hero + Sidebar** — koko leveyden kuvabaneri, sisältö kahdessa sarakkeessa (pääsisältö + tietopalkki)
- **C: Välilehdet** — profiilin otsikkorivi ylhäällä, sisältö jaettuna välilehtiin (Perustiedot / Kilpailut / Sukutaulu / Kuvat)

## Mitä katsoa
- Tuntuuko lineaarinen (A) johdonmukaiselta vai tuleeko liikaa scrollaamista?
- Toimiiko sidebar (B) — onko tietopalkki oikealla paikallaan vai häiritsevä?
- Onko välilehtirakenne (C) selkeä käyttää — löytääkö kilpailutulokset helposti?
- Miltä sukutaulu näyttää kussakin layoutissa?
