# Sketch 011 — Ajankohtaista etusivulla

**Designkysymys:** Missä ja miten uusin blogipostaus näytetään etusivulla?

**Tunnelma:** Eläväinen, lämmin — rustiikkinen talli-estetiikka.  
**Ydintehtävä:** Käyttäjä lukee lyhyen ingressin → klikkaa "Lue lisää".  
**Kentät:** otsikko, päiväys, sisältöteksti (ingressi).

## Variantit

| Variantti | Kuvaus | Vahvuus | Heikkous |
|-----------|--------|---------|---------|
| A | Uutinen kolmantena overlay-korttina gridissä | Yhtenäinen, ei riko rakennetta; käyttäjä näkee uutisen samalla tasolla tilastojen kanssa | Kortti ahdas pitkälle tekstille; ei niin "editoriaalinen" |
| B | Erillinen leveä uutispalkki hero:n jälkeen | Näkyvä, tilaa ingressille; jakaa sivun selkeisiin osioihin | Lisää scrollattavaa; vahvempi visuaalinen painotus kuin tilastot |
| C ★ | Editoriaalinen nostokortti .frontpage-uutinen-alueella | Pehmeä, lämmin; käyttää olemassa olevaa 2-kolumni-rakennetta; selkein muutos koodiin | Hierarkkisesti alapuolella — käyttäjä saattaa ohittaa |
