# Sketch 005: Etusivu

## Designkysymys
Mikä rakenne toimii parhaiten etusivulle, jossa on iso hero, tilastot, tallin esittely, uutinen ja linkki kasvatukseen?

## Miten avaat
`open .planning/sketches/005-etusivu/index.html`

## Variantit
- **A: Tilastopalkki** — Hero → kultainen tilastopalkki (hevoset/varsat/vuosi) → esittely + kasvatus-CTA rinnakkain → uutinen. Selkeä hierarkia, tilastot erottuvat heti.
- **B: Rinnakkain** — Hero CTA-painikkeilla → esittely + tilastokortit rinnakkain → uutinen + kasvatus-CTA rinnakkain. Tasapainoisempi, enemmän sisältöä kerralla.
- **C: Overlay-kortit** — Hero koko ruutu → kolme korttia kelluu herolle päälle (tilastot + uutinen) → esittely + uutinen alla. Visuaalisesti rohkein.

## Mitä katsoa
- Missä variantissa tilastot (hevosmäärä, varsoja) erottuvat parhaiten?
- Sopiiko overlay-korttityyli (C) rustiikkiseen tallitunnelmaan?
- Onko B:n kaksisarakeinen rakenne selkeä vai sekava?
- Missä kasvatus-linkki löytyy luontevimmin?
