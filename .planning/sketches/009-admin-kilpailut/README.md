# 009 — Admin Kilpailut

**Designkysymys:** Miten kilpailutulokset esitetään — taulukkomaisesti vai aikajana-formaatissa?

**Variantit:**
- **A: Kompakti lista + modal** — Kompakti taulukkorivi (nimi, päivämäärä, sijoitus, pisteet), toiminnot laajentuvat. Uusi kilpailu modaalissa.
- **B: Aikajana** — Kilpailut kronologisesti vuosittain ryhmiteltynä. Visuaalisempi, tarina-henkinen. Sijoitusmedaljit korostuvat.

**Voittaja:** A: Kompakti lista + modal ★

**Tagit:** admin, kilpailut, competitions, timeline, compact-list
