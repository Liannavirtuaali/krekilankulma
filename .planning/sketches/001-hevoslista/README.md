# Sketch 001: Hevoslista

## Designkysymys
Mikä korttiformaatti tuntuu oikealta virtuaalitallin hevoslistauksessa?

## Miten avaat
`.planning/sketches/001-hevoslista/index.html` — avaa selaimessa

## Variantit
- **A: Grid-kortit** — tuttu 3-4 sarakkeen ruudukko, kuva ylhäällä, lajibadge
- **B: Listakortit** — vaakasuuntainen kuva + teksti, mahtuu kuvaus per hevonen
- **C: Mosaiikki** — vaihteleva ruudukko jossa featured-hevonen isompana

## Mitä katsoa
- Tuntuuko ruudukko (A) riittävän informatiivilta ilman kuvausta?
- Antaako listanäkymä (B) liian paljon tekstiä — vai onko se parempi sivustoille joilla on paljon kuvauksia?
- Toimiiko mosaiikki (C) ilman oikeita kuvia? Näyttääkö se erikoiselta vai mielenkiintoiselta?
- Miltä suodatuspainikkeet tuntuvat?
