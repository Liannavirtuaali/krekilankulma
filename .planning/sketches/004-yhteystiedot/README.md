# Sketch 004: Yhteystiedot-sivu

## Designkysymys
Miten esitellä tallinomistajan yhteystiedot (nimimerkki, VRL-tunnus, sähköposti) niin, että sähköpostiin pääsee helposti ja sivu tuntuu lämpimältä?

## Miten avaat
`open .planning/sketches/004-yhteystiedot/index.html`

## Variantit
- **A: Korttikortti** — Yksinkertainen kortti keskellä, ikonit vasemmalla, mailto-painike alhaalla. Puhdas ja selkeä.
- **B: Jaettu layout** — Kaksi palstaa: vasen on tervetuloteksti + nimimerkki/VRL, oikea on yhteystietolista. Enemmän tilaa esittelylle.
- **C: Kirjemainen** — Kirjeen muoto, tummalla otsikkoalueella tallin nimi ja lämmin johdanto. Persoonallisin vaihtoehto.

## Mitä katsoa
- Löytyykö sähköpostiosoite helposti kaikissa varianteissa?
- Miten VRL-tunnus näyttää — erottuuko se tarpeeksi tekniseksi tiedoksi?
- Sopiiko kirjemainen tyyli (C) rustiikkiseen talli-tunnelmaan?
- Onko B:n kaksisarakkeinen layout tarpeellinen näin vähälle sisällölle?
