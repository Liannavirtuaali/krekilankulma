## Interaction Contracts

### Navigation active state
- Current page link gets `class="active"` — detected by `basename($_SERVER['PHP_SELF'])`
- Active + hover style: color `#e8d5b7`, `text-decoration: underline`

### Horse card links
- Card h3 link: `color: #3d2b1f; text-decoration: none`
- Hover: `text-decoration: underline`
- Links resolve via `horseUrl($horse)` helper (slug-based URL preferred, fallback to `?id=`)

### External pedigree links
- EVM horses with `profile_url`: open `target="_blank" rel="noopener"`
- URL validated with `filter_var(FILTER_VALIDATE_URL)` before output; falls back to `#` on invalid URL

### 404 handling
- Missing horse or `is_deleted = 1`: `http_response_code(404)`, friendly Finnish message, `exit`
- Message: "Hevosta ei löydy tai se on poistettu." (from hevonen.php)
- No stack traces or DB structure exposed

### No forms in this phase
- All pages are read-only. No POST actions, no CSRF tokens needed.
- Yhteystiedot has a `mailto:` link only.

## XSS Output Contract

Locked decision D-02: all dynamic values use `e()` helper (wraps `htmlspecialchars()`).

| Context | Rule |
|---------|------|
| All DB-sourced text | `e($value)` |
| Multi-line text (description, pedigree_notes) | `nl2br(e($value))` |
| URLs from DB (profile_url) | `filter_var(FILTER_VALIDATE_URL)` validation first, then `e()` |
| Integer values | Cast to `(int)` or `(string)` before `e()` |
| Constants (SITE_NAME, SITE_URL) | `e(SITE_NAME)` — treated as untrusted for consistency |
