<task type="auto">
  <name>Task 1: Kilpailujen hallinta (competitions.php)</name>
  <files>public/admin/competitions.php</files>
  <action>
Luo `public/admin/competitions.php`.

1. Require `../src/includes/db.php`, kutsu `requireLogin()`.
2. `$horse_id = (int)($_GET['horse_id'] ?? 0)`.
3. Hae hevonen: `SELECT id, name FROM horses WHERE id = :id AND is_deleted = 0` — jos ei löydy, ohjaa `horses.php`.
4. CSRF-token sessioon jos ei ole.
5. Hae `$edit_id = (int)($_GET['edit'] ?? 0)` — muokkaustila.

**POST-käsittely — toiminto tunnistetaan `$_POST['action']`-kentästä:**

- `action=add`:
  - Tarkista CSRF.
  - Kerää: `competition_name` (pakollinen), `competition_date` (date), `placement` (text), `points` (int/null), `notes` (textarea).
  - `INSERT INTO competitions (horse_id, competition_name, competition_date, placement, points, notes) VALUES (...)`.
  - Ohjaa `competitions.php?horse_id={id}&added=1`.

- `action=edit`:
  - Tarkista CSRF.
  - `$comp_id = (int)($_POST['comp_id'] ?? 0)`.
  - Varmista omistajuus: `SELECT id FROM competitions WHERE id = :comp_id AND horse_id = :horse_id`.
  - `UPDATE competitions SET competition_name=..., competition_date=..., placement=..., points=..., notes=... WHERE id = :comp_id`.
  - Ohjaa `competitions.php?horse_id={id}&updated=1`.

- `action=delete`:
  - Tarkista CSRF.
  - `$comp_id = (int)($_POST['comp_id'] ?? 0)`.
  - Varmista omistajuus kuten yllä.
  - `DELETE FROM competitions WHERE id = :comp_id`.
  - Ohjaa `competitions.php?horse_id={id}&deleted=1`.

**HTML-näkymä:**
- Otsikko: "Kilpailut — [hevosen nimi]"
- Kilpailulista taulukossa: Kilpailu, Päivämäärä, Sijoitus, Pisteet, Muistiinpanot, Toiminnot.
- Toiminnot: "Muokkaa" (linkki `?horse_id={id}&edit={comp_id}`) ja "Poista" (POST-lomake action=delete).
- Jos `$edit_id > 0` ja vastaava kilpailu löytyy, näytä esitäytetty muokkauslomake lisäyslomakkeen sijaan (tai alla).
- Lisäyslomake: kentät, CSRF, action=add, horse_id piilokenttä.
- Linkki takaisin horses.php:lle.
- Sisällytä admin_header.php (`$pageTitle = 'Kilpailut'`) ja admin_footer.php.
  </action>
  <verify>
    <automated>php -l public/admin/competitions.php</automated>
  </verify>
  <done>competitions.php listaa kilpailut. Lisäys, muokkaus ja poisto toimivat. Omistajuustarkistus estää toisten hevosten kilpailujen muokkaamisen.</done>
</task>

<task type="auto">
  <name>Task 2: Kasvatustietojen hallinta (foals.php)</name>
  <files>public/admin/foals.php</files>
  <action>
Luo `public/admin/foals.php`.

1. Require `../src/includes/db.php`, kutsu `requireLogin()`.
2. `$horse_id = (int)($_GET['horse_id'] ?? 0)`.
3. Hae hevonen: `SELECT id, name FROM horses WHERE id = :id AND is_deleted = 0` — jos ei löydy, ohjaa `horses.php`.
4. Hae dropdown-listat:
   - `SELECT id, name FROM horses WHERE is_deleted = 0 ORDER BY name` — sire_id ja dam_id varsamerkinnöille.
5. CSRF-token sessioon jos ei ole.
6. Hae `$edit_id = (int)($_GET['edit'] ?? 0)` — muokkaustila.

**POST-käsittely — `$_POST['action']`:**

- `action=add`:
  - Tarkista CSRF.
  - Kerää: `foal_name` (pakollinen, text), `sire_id` (int/null), `dam_id` (int/null), `birth_year` (int, 4 numeroa), `gender` (select: ori/tamma/ruuna/tuntematon), `status` (select: expected/born).
  - `INSERT INTO foals (horse_id, foal_name, sire_id, dam_id, birth_year, gender, status) VALUES (...)`.
  - Ohjaa `foals.php?horse_id={id}&added=1`.

- `action=edit`:
  - Tarkista CSRF.
  - `$foal_id = (int)($_POST['foal_id'] ?? 0)`.
  - Varmista omistajuus: `SELECT id FROM foals WHERE id = :foal_id AND horse_id = :horse_id`.
  - `UPDATE foals SET foal_name=..., sire_id=..., dam_id=..., birth_year=..., gender=..., status=... WHERE id = :foal_id`.
  - Ohjaa `foals.php?horse_id={id}&updated=1`.

- `action=delete`:
  - Tarkista CSRF.
  - `$foal_id = (int)($_POST['foal_id'] ?? 0)`.
  - Varmista omistajuus.
  - `DELETE FROM foals WHERE id = :foal_id`.
  - Ohjaa `foals.php?horse_id={id}&deleted=1`.

**HTML-näkymä:**
- Otsikko: "Kasvatus — [hevosen nimi]"
- Varsalista taulukossa: Varsan nimi, Isä, Emä, Syntymävuosi, Sukupuoli, Status, Toiminnot.
- Toiminnot: "Muokkaa" (linkki `?horse_id={id}&edit={foal_id}`) ja "Poista" (POST-lomake).
- Jos `$edit_id > 0`, näytä esitäytetty muokkauslomake.
- Lisäyslomake: kaikki kentät, sire_id ja dam_id selekteinä (hevoslista), status-selekti ("Odotettu" = expected, "Syntynyt" = born).
- Linkki takaisin horses.php:lle.
- Sisällytä admin_header.php (`$pageTitle = 'Kasvatus'`) ja admin_footer.php.
